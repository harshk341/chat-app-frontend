# Real-time Chat Application

A one-to-one real-time messaging system featuring secure JWT authentication and persistent message history.

---

## Features

- **Real-time Communication**: Instant message delivery.
- **Secure Access**: JWT-based authentication.
- **Persistence**: All messages are saved and retrievable.

---

## Tech Stack

- **React** 
- **TypeScript** 
- **Express.js**
- **Socket.io**
- **MongoGB**

---

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/harshk341/chat-app-frontend.git

2. Navigate to the project directory:
   ```bash
   cd chat-app-frontend

3. Install dependencies:
   ```bash
   yarn install
   ```
   or
   
   ```bash
   npm install

4. Start the development server:
   ```bash
   yarn dev
   ```
   or
   
   ```bash
   npm run dev

5. Open your browser and visit `http://localhost:5173`
